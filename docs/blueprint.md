# **App Name**: Save & Fly

## Core Features:

- Destination Image Retrieval: Automatically fetch representative images from Google Images for recommended destinations to enhance user interest.
- UI for Travel Input: A clean interface to input travel parameters: departure, destination, dates, and traveler count. Includes flexible date options and a search button.
- Data Aggregation: Collect flight data from multiple sources to create a comprehensive database. Start with a simplified single source if needed.
- Intelligent Sorting Engine: Sort flight options from various sources and filter top 20 results
- Booking Redirection: Redirect users to the official website or OTA to complete the booking process safely. Integrates search parameters into the redirect URL.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke a sense of calmness and trustworthiness, aligning with travel and exploration.
- Background color: Very light blue (#E3F2FD), a desaturated tint of the primary color for a clean and unobtrusive backdrop.
- Accent color: Pale green (#AED581), an analogous color to blue, to highlight key interactive elements like 'Book Now' buttons.
- Headline font: 'Poppins', sans-serif, for headlines.
- Body font: 'PT Sans', sans-serif, for body text.
- Use clean, modern icons for travel-related categories. Icons should match overall simple style of UI.
- A grid-based layout to present flight options in a structured manner.
- Smooth transitions and loading animations when fetching flight data.
